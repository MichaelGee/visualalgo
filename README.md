## Docker

`docker build . --tag="visualalgo:latest"`

`docker run -d --name visualalgo -p 3000:3000 sieve:latest`

Open your browser and navigate to http://localhost:3000

## Development

`npm intall`

`npm run dev`

or

`yarn`

`yarn dev` 

Open your browser and navigate to http://localhost:3000

## Testing

`npm run test`
